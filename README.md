# 👉 CyberFusion‑XDR  
### Open‑Source SOC Analytics & Threat Detection Framework

CyberFusion‑XDR is a fast, modular, developer‑friendly XDR platform built for analysts, engineers, and researchers who need a clean, extensible, and secure environment for cybersecurity analytics.  
It combines Python, Dash, Docker, and a full DevSecOps pipeline (CI + Bandit + Semgrep + Gitleaks) to deliver a modern, production‑ready workflow for threat detection, risk scoring, log analysis, and MITRE ATT&CK intelligence.
